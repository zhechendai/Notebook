该Notes利用GitBook，并使用VSCode编辑器搭配Markdown语言编辑。

使用GitBook Version 2.2.0, 是很久远的版本了，但我喜欢目录栏绿色的阅读小勾，能清晰展现阅读状态和进度，该页面使用的gitbook-plugins，如下

"plugins": [
        "splitter",
        "auto-scroll-table",
        "katex"
    ]

其中数学公式使用Latex语言编辑，使用katex插件展示，例如

{% math %}\int_{-\infty}^\infty e^{-x^2} = \sqrt{\pi}{% endmath %}

显示结果为

∫−∞∞e−x2=π \int_{-\infty}^\infty e^{-x^2} = \sqrt{\pi} ∫​−∞​∞​​e​−x​2​​​​=√​π​​​
